Welcome to CS2D's documentation!
===================================

This repository contains (1) code for the paper CS2D: A Novel Framework for Backdoor Learning and (2) implementations for several state-of-the-art frameworks for backdoor learning.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::
